# 📘 Data Types in Python

This folder contains Python programs to understand the basic Data Types in Python.

## 📚 Topics Covered

- 🔢 Numbers
- 🔤 Strings
- 📋 Lists
- 📦 Tuples
- 🧩 Sets
- 📖 Dictionaries
- ✅ Boolean
- 🔍 Type Checking
- 🔄 Type Conversion
- 📝 Practice Questions

---

## 📂 Files

| File Name | Description |
|-----------|-------------|
| 01_numbers.py | Learn about Integer, Float and Complex Numbers |
| 02_strings.py | Learn about String Data Type |
| 03_lists.py | Learn about List Data Type |
| 04_tuples.py | Learn about Tuple Data Type |
| 05_sets.py | Learn about Set Data Type |
| 06_dictionaries.py | Learn about Dictionary Data Type |
| 07_boolean.py | Learn about Boolean Data Type |
| 08_type_checking.py | Check Data Types using `type()` |
| 09_type_conversion.py | Convert one Data Type into another |
| 10_practice_questions.py | Practice Questions for Data Types |

---

## 🎯 Learning Outcomes

After completing this folder, you will be able to:

- Understand all basic Python Data Types.
- Differentiate between List, Tuple, Set, and Dictionary.
- Check the type of any variable using `type()`.
- Convert one Data Type into another.
- Build a strong foundation for advanced Python programming.
